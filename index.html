Ellipse vs Hyperbola — The Key Memory Trick
Ellipse: \(c^2 = a^2 - b^2\) → b² is subtracted (INSIDE the ellipse)
Hyperbola: \(c^2 = a^2 + b^2\) → b² is added (OUTSIDE, going away)
Think: Ellipse = enclosed → minus. Hyperbola = open → plus.
⭕ Circle: x²+y²=r²
📐 Parabola: y²=4ax (opens right)
🥚 Ellipse: x²/a²+y²/b²=1 (a>b)
📡 Hyperbola: x²/a²−y²/b²=1

3×3 Determinant (Cofactor Expansion)
Must Know — Expand along Row 1
\[\det\begin{pmatrix}a&b&c\\d&e&f\\g&h&i\end{pmatrix} = a(ei-fh) - b(di-fg) + c(dh-eg)\]
🟠
Sign Pattern for Cofactors
\(\begin{pmatrix}+&-&+\\-&+&-\\+&-&+\end{pmatrix}\) — alternating checkerboard of +/− signs. Top-left is always +.
🟠
Sarrus' Rule (Visual Shortcut for 3×3)
Copy first two columns to the right. Sum of 3 downward diagonals MINUS sum of 3 upward diagonals.

⚠️ Most Common Mistakes ▾
🔴
Log of Sum ≠ Sum of Logs
\(\log(a+b) \neq \log a + \log b\). Only \(\log(ab) = \log a + \log b\)
🔴
Ellipse vs Hyperbola c²
Ellipse: \(c^2=a^2-b^2\) (minus). Hyperbola: \(c^2=a^2+b^2\) (plus). Swapping these = wrong answer!
🔴
Vieta's Sum Sign
\(\alpha+\beta = -b/a\) NOT \(+b/a\). The negative sign is always there!
🔴
General Form of Circle: Missing the 2
In \(x^2+y^2+2gx+2fy+c=0\), center is \((-g,-f)\). Don't forget the coefficient 2 when reading off g and f.
🔴
cos 2A Has Three Forms
\(\cos 2A = \cos^2A-\sin^2A = 1-2\sin^2A = 2\cos^2A-1\). Picking the wrong form wastes time.
🔴
i² = −1, NOT +1
Always: \(i^2=-1\). A very common slip that flips the entire answer.
🔴
Perpendicular Slope
If slope = m, perpendicular slope = \(-1/m\). NOT just \(1/m\).
🔴
Inverse Matrix Condition
Matrix has no inverse if det = 0 (singular). Always check det before inverting.
🔴
Binomial: r starts at 0
General term is \(T_{r+1}\) with \(r=0,1,2,\ldots,n\). If asked for 3rd term, use \(r=2\), not \(r=3\).
🔴
Circular Permutation
\((n-1)!\) for arranging n objects in a circle, NOT \(n!\). The extra division accounts for rotational symmetry.
